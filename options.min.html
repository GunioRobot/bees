Inline Block Module

<div class="block">
     <div>...</div>
     <div>...</div>
     <div>...</div>
</div>

This is when you want something like a matrix with pieces of similar-width content sitting together when possible, then wrapping as needed. The divs immediately within the block div are required wrappers; anything else goes.

No widths and heights are necessary on any divs, unless these are filled with long strings of text, in which case widths should be assigned to the enclosing divs, lest they expand across the page.

Currently the default alignment is bottom, but this can be changed to: top, middle, or bottom.

<div class="block middle">
     <div>...</div>
</div>

Auto-expanding tables emulate html tables (with all the div wrappers to be expected!) Cells can individually be given horizontal and vertical alignment options: left, center, right, top, middle, bottom.
