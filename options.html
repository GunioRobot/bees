Animation Module

<div class="beeFlip">
    <div class="flip front">
        ...
    </div>
    <div class="flip back">
        ...
    </div>
</div>

Note this relies on absolute positioning; for the rest of the page to flow, height must be known. In this case, adding a margin-bottom of 170px to beeFlip accommodates flip content (150px) plus padding (20px).

Shadow & Rounded Corner Modules


<img class="shadow round8" /> <div class="shadow round20">...</div>

Shadows and Rounded Corners combine easily. Rounded corner presets are: 5px, 8px, 10px, 15px, and 20px.

Note that combining Shadows with Animations can be issue-prone. In Firefox, the flip back shows through, whereas in WebKit, this is not so. This means that in Firefox, any shadows are doubled when compared to WebKit.

Inline Block Module

<div class="block">
     <div>...</div>
</div>
<div class="middle block">
     <div>...</div>
</div>

This is when you want something like a matrix with pieces of similar-width content sitting together when possible, then wrapping as needed. Currently the default alignment is bottom, but this can be changed to: top, middle, or bottom.
