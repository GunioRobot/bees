HTML5 Elements and CSS3 Flexible Box Layout

Compatibility: Chrome 5+, Safari 3+, Firefox 3+

Libraries: Prototype 1.7rc3, Scripty 2b1, Quirksmode (browser detect), HTML5-boilerplate (CSS reset), BeeStyle

BeeStyle Components: button, animation, glow, scroll (animation and scroll only supported by Webkit currently)

Javascript is required to fix a bug in Firefox's current CSS3 Flexible Box Layout implementation (flex-scroll.js)

Javascript is also required for scrolling capabilities within the layout areas (rather than scrolling the page vertically and losing the footer) (flex-scroll-2.js)

This customizable fluid layout targets any screen size, including iPhone 4, older iPhone, and iPad resolutions (todo: iPhone relayout)

Width and height are calculated automatically either by CSS/JS (Webkit) or JS (Gecko), even when optional html exists

Sidebars flex depending on screen size in relation to center content (can be set to fixed width if desired)

Issues: not working with Modernizr currently; no IE or Opera compatibility
