Usage & Dependencies

First include HTML5-boilerplate and (optionally) WebKit scrollbars

<link rel="stylesheet" href="stylesheets/reset.css">
<link rel="stylesheet" href="stylesheets/scroll.css">

Then include the core BeeStyle-BeeFlex stylesheet

<link rel="stylesheet" href="stylesheets/beestyle-beeflex-a1.min.css">

Then load up Modernizr and Prototype

<script src="javascripts/modernizr-1.7.min.js"></script>
<script src="javascripts/prototype-1.7.js"></script>

Then include the core Beestyle-BeeFlex javascript

<script src="javascripts/beestyle-beeflex-a1.min.js"></script>

Note this page includes demo javascript to power the extra/color toggles

<script src="javascripts/beestyle-toggle-demos.min.js"></script>
