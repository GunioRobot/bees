Quick Setup (for fresh new pages)

Download this zip file, which includes a sample page and all dependencies

Integrate into existing pages (usage & dependencies)

Required assets are noted in bold

First include HTML5-boilerplate and (optionally) Webkit scrollbars

<link rel="stylesheet" href="stylesheets/reset.css">
<link rel="stylesheet" href="stylesheets/scroll.css">

If including Webkit scrollbars, ensure these images present in:

images/scroll

down.png left.png right.png up.png

Then include the core BeeStyle-BeeFlex stylesheet

<link rel="stylesheet" href="stylesheets/beestyle-beeflex-a1.min.css">

Then load up Modernizr and jQuery

<script src="javascripts/modernizr-2.0.min.js"></script>
<script src="javascripts/jquery-1.6.min.js"></script>

Then include the core Beestyle-BeeFlex javascript

<script src="javascripts/beestyle-beeflex-a1.min.js"></script>

Note this page includes demo javascript to power the demo toggles

<script src="javascripts/beestyle-demos.min.js"></script>
