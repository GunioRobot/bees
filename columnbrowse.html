Add as many columns as needed by adding <section>s inside a <div> with a class of "beeColumnBrowse" Any content not included in the column browser should be enclosed in <div>s with a class of "beeColumnExclude"

Note that this Column Browser should not be used alongside the Multi-Column module. The Multi-Column module lets content flow naturally through columns (but is not yet supported fully, spec-wise). The Column Browser restricts given content to a given column, so there is no reflowing of content between columns. The Column Browser is intended to function more like the Mac OS X finder.
